# HERO.IO

[Live Demo](https://hero-io-by-kabir-devx.netlify.app/)

## Description
Hero.io is a web app that showcases trending apps. Users can explore app details, view ratings and downloads, install apps, and track installed apps. It includes sorting options, interactive charts, and toast notifications for real-time feedback, all within a responsive, modern interface.

## Technologies
- React 19  
- TailwindCSS 4  
- DaisyUI  
- React Router 7  
- Axios  
- Recharts  
- React Icons  
- React Toastify  
- Vite
